# HI Shell Detection With Standardized PV Slices

This checkout is prepared for GitHub around the public project directory:

```text
hishells-pv-research/
```

That directory contains the source code, small catalog tables, configs,
documentation, result summaries, and figures for the HI shell PV-slice
detection project. Local experiment outputs, raw FITS cubes, generated arrays,
model checkpoints, and scratch files are ignored at the parent level.

Start in `hishells-pv-research/` for installation and run instructions.

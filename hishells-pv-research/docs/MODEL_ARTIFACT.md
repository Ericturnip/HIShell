# Published CNN Checkpoint

The public checkpoint for the paper-facing clean physical baseline is:

```text
artifacts/models/hi_shell_pv_unet_clean_physical_baseline.keras
```

This is the high-recall Keras model from run `pv_unet_clean_physical_baseline_20260522_022751`. The model is a 2D U-Net/CNN mask segmenter trained on standardized PV slices with shape `(96, 256, 1)`, a 5 kpc spatial window, and a 200 km/s velocity window.

The companion metadata file is:

```text
artifacts/model_metadata.json
```

It records the checkpoint SHA-256 checksum, source run, recommended probability threshold (`0.075`), and the headline validation, test, stress, and DDO 53 physical-grid metrics. The raw evaluation JSON files and historical run artifacts are intentionally omitted.

Inference example:

```bash
python -m src.infer.infer_pv \
  --config training_data/standardized_5kpc_200kms_clean_physical_baseline/train_standardized_high_recall.yaml \
  --model artifacts/models/hi_shell_pv_unet_clean_physical_baseline.keras \
  --out runs/published_model_inference
```

The config in that example is generated by `scripts/run_clean_physical_baseline.sh`; raw FITS cubes and generated PV arrays are intentionally not stored in Git.
